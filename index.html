Symbole sur Mac

Les raccourcis clavier vous permettent d'utiliser des combinaisons de touches au lieu d'une souris, d'un trackpad ou d'un autre périphérique d'entrée pour effectuer des opérations spécifiques. Pour utiliser les raccourcis clavier, appuyez longuement sur une ou plusieurs touches de modification, puis appuyez sur la dernière touche de la touche de raccourci.
Par exemple, pour utiliser Commande + C (copier), maintenez la touche Commande enfoncée, puis appuyez sur C, puis relâchez les deux touches.

Sur les menus et les claviers des ordinateurs Mac, les touches (y compris les touches de modification) sont généralement représentées par des symboles :

Commande du Clavier Fonction
Commande (cmd) ⌘
Maj ⇧
Option ou Alt ⌥
Contrôle ou Ctrl ⌃
Verrouillage majuscule ⇪

Symbole sur Windows

Les raccourcis clavier vous permettent d'utiliser des combinaisons de touches au lieu d'une souris ou d'un autre périphérique d'entrée pour effectuer des opérations spécifiques. Pour utiliser les raccourcis clavier, appuyez longuement sur une ou plusieurs touches de modification, puis appuyez sur la dernière touche du raccourci. Par exemple, pour utiliser Ctrl + C (copier), maintenez la touche Ctrl enfoncée, puis appuyez sur C, puis relâchez les deux touches.

Sur les menus et les claviers des ordinateurs Windows, les touches (y compris les touches de modification) sont généralement représentées par des symboles :

Les raccourcis les plus courants sur Mac

  • ⌘ + C : Copier
  • ⌘ + X : Couper
  • ⌘ + V : Coller
  • ⌘ + Z : Annuler la commande
  • ⌘ + A : Sélectionner tout
  • ⌘ + F : Rechercher
  • ⌘ + G : Rechercher l’occurrence suivante
  • ⌘ + N : Ouvrir une nouvelle fenêtre ou un nouveau document
  • ⌘ + S : Enregistrer un document
  • ⌘ + P : Imprimer un document
  • ⌘ + Maj + 3 : Capture d’écran entier
  • ⌘ + Maj + 4 : Capture d’écran avec curseur de sélection
  • ⌘ + W : Fermer la fenêtre active
  • ⌘ + Option + W : Fermer toutes les fenêtres
  • ⌘ + Maj + Tilde (~) : Passer à la fenêtre suivante la plus récemment utilisée dans l’application active
  • ⌘ + H : Masquer les fenêtres de l'application ouverte au premier plan

  • ⌘ + Option + H : Masquer les fenêtres de toutes les applications ouvertes, hormis celles de l’application au premier plan
  • ⌘ + Q : Quitter une application
  • ⌘ + Option + Echap : Forcer la fermeture d’une application
  • ⌘ + Tab : Revenir à l’application ouverte la plus récemment utilisée
